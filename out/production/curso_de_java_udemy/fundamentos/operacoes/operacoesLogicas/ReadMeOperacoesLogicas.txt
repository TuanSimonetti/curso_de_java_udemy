Operadores Lógicos

Os operadores lógicos possuem ligação direta com o tipo Boolean. Nesse contexto, estipulamos o procedimento visando se
a situação é propicia (True) ou não (False).


TABELA VERDADE DO "E":

Sempre que tiver uma expressão falsa dentro do "&&", o resultado será falso. Porque é necessário que ambas as
condições sejam verdadeiras (True) para que o resultado seja verdadeiro (True).

Exemplo:

Utilizando o "&&" podemos unir duas situações onde:

Temos Dinheiro? True
Temos Sol? True
Podemos ir ao Shopping? True

Temos Dinheiro? True
Temos Sol? False
Podemos ir ao Shopping? False

Temos Dinheiro? False
Temos Sol? True
Podemos ir ao Shopping? False

Temos Dinheiro? False
Temos Sol? False
Podemos ir ao Shopping? False


OBS.:




TABELA VERDADE DO "OU" (||)

Nesse caso, funciona ao contrário do "E" (&&). Onde sempre que tiver uma situação que contiver um valor verdadeiro (True)
o resultado será True. Quando todos os resultados forem falsos, o resultado final será false (False).

Exemplo:

Temos Dinheiro? True
Temos Sol? True
Podemos ir ao Shopping? True

Temos Dinheiro? True
Temos Sol? False
Podemos ir ao Shopping? True

Temos Dinheiro? False
Temos Sol? True
Podemos ir ao Shopping? True

Temos Dinheiro? False
Temos Sol? False
Podemos ir ao Shopping? False
