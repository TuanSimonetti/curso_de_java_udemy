Ao todo são 8 tipos primitivos:

* Tipos Númericos:
- Byte: 8 bits, que corresponde à <128 até >127;
- Shot: 2 bytes;
- Int: 4 bytes;
- Long: 8 bytes;
- Float: 4 bytes;
- Double: 8 bytes.

OBS.1: Byte, short, int e long são tipos inteiros;
OBS.2: Float e Double são tipos com ponto flutuante (descimal).

* Tipos de texto:
- char: um único caracter, como: 'a', 's', 'n', '', '?', '*', '1', etc;
- boolean: True or False (verdadeiro ou falso).

O java não considera o valor, mas a tipagem. Logo, se escrevermos "double a = 3;", o Java irá considerar o valor como
Int e irá reclamar, pedindo para converter em "Double".
